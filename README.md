# Gaze Tracking to Analyze People's Attention (APP Project)

## Installation

Create python environnement (here is an example using conda):
```bash
conda create -n APP python=3.8 -y
conda activate APP
```

Install the required packages:
```bash
pip install -r requirements.txt
```

## Project Structure

data/: Directory containing the dataset.
src/: Source code for data loading and processing.
gaze_demo.py: Example script to demonstrate gaze data loading and processing.
